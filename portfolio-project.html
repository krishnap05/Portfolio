Personal Portfolio Website

A static portfolio built with HTML, CSS, JavaScript and SVG

Overview

This site showcases my projects, skills, and contact information. It uses a responsive layout, animated SVG backgrounds, per-character headline reveals, and small JS interactions for parallax and scroll behaviour.

Screenshots

To add a visual screenshot for this project, place an image named portfolio-1.png into assets/screenshots/. Recommended width: ~1200px for clarity. Supported formats: PNG, JPG.

Example path: assets/screenshots/portfolio-1.png

When you place the file there, open this page (or refresh) to see it — you can then edit the markup below to include the image if you want a thumbnail:

<figure>
  <img src="assets/screenshots/portfolio-1.png" alt="Portfolio screenshot" />
  <figcaption>Homepage — dark hero and projects</figcaption>
</figure>
