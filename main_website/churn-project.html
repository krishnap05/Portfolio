This interactive dashboard predicts customer churn using machine learning. It analyzes telecom data to identify churn patterns and visualize key drivers.

How to Run the Dashboard

To view the dashboard locally:

  1. Clone the repository from GitHub
  2. Open your terminal and navigate to the project folder
  3. Run streamlit run app.py

This will launch the dashboard in your browser at localhost:8501.
